Якийсь текст з абзайом

  • git clone path-to-repository
  • git status - статус текущего локального резозитория
  • git commit - am "added new file" - зафиксировать изменения и дать им коментарий
  • git push - отправить все закомисенниые изминения в репозиторий на сервер
  • git pull - забрать свежие обновление
